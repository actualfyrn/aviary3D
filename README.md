An attempt at a relatively simple, lightweight, multi-platform, Rust-based 3D-first game and software engine.

APIs required or used:  
- wgpu (graphics)  
- cpal (audio)  
- rapier3d-f64 (rigid body physics)  
- audionimbus (later, for HRTF audio)  
- Valve's GameNetworkingSockets (later, for online connectivity)
